\smallskip
\noindent СЛУЧАЙНЫЙ ПРОЦЕСС, СТОХАСТИЧЕСКАЯ СВЯЗЬ, ОЦЕНКА ПАРАМЕТРОВ, КОПУЛА.

\smallskip
Выпускная работа бакалавра посвящена применению копул к изучению связей между оценками параметров моделей случайных процессов. Оценки параметров моделей этих процессов были получены ранее с помощью алгоритма интервального и точечного оценивания. Исследование интервальных оценок, доставляемых алгоритмом, позволяет делать выводы о его эффективности. В рамках работы получены результаты, позволяющие уменьшить количество исследований, требуемых для улучшения реализации алгоритма интервального оценивания.

\end{abstract}
